Debugging Tips

If Tests Fail:

  1. Clear Browser Cache
    • Chrome/Edge: Ctrl + Shift + Delete
    • Or use Hard Refresh: Ctrl + Shift + R
    • Or try Incognito/Private Mode
  2. Check JavaScript Console
    • Press F12 to open DevTools
    • Go to Console tab
    • Look for any red error messages
  3. Verify File Versions
    • Check that main.js is loading (Network tab)
    • Verify timestamp is recent
    • Check file size (~8KB)
  4. Check URL Parameters
    • After clicking category, check address bar
    • Should show: ?category=CategoryName
    • If missing, category filtering isn't working
